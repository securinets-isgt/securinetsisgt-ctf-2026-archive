# Frontend Export

This folder contains the public-safe frontend/theme side of the CTFd project.

Contents:

- `theme-hacker/`
  - The custom Batman/Riddler CTFd theme source and built static assets
- `standalone-pages/`
  - Standalone HTML pages kept as references
- `docs/`
  - A static page you can publish with GitHub Pages

GitHub Pages note:

- GitHub Pages cannot run the real CTFd platform
- It can only host static files such as the `docs/` page or theme assets
- If the club wants a public archive/showcase repo, publish `frontend/docs/`
