# Frontend Export

This folder contains the public-safe frontend/theme side of the CTFd project.

Contents:

- `theme-hacker/`
  - The custom Batman/Riddler CTFd theme source and built static assets
- `standalone-pages/`
  - Standalone HTML pages kept as references

GitHub Pages note:

- GitHub Pages cannot run the real CTFd platform
- It can only host static files
- This folder is mainly for the theme/frontend source and static assets
