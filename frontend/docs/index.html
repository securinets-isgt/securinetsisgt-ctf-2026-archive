Securinets ISGT CTF Archive

This public archive contains the custom frontend theme and a safe code snapshot of the CTF platform. The live backend is hosted separately and is not served by GitHub Pages.

Frontend

The Batman/Riddler visual theme, templates, SCSS, built static assets, and showcase pages are included in frontend/theme-hacker/.

Backend

The safe backend code snapshot lives in backend/. Runtime data, uploads, databases, and secrets were intentionally removed.
